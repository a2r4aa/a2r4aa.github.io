B站手機客戶端越來越臃腫了，低質量視頻也有點氾濫的樣子，所以有了寫一個第三方客戶端的想法，本來對 React 瞭解一些，選用 React Native 作爲開發語言的，結果寫到一半之後，處理 ListView 頁面滾動時，有無法快速滾動的問題，所以我就轉向了 Flutter。Dart 這門語言挺像Java的，所以上手難度要比 React Native高一些，經過了一個月的磕磕絆絆，我才寫出了這個Demo







註：僅支持 Android 9 及以上版本設備

使用到的權限有

  • 連接互聯網

  • 讀取手機存儲空間（可選）

完成了一下功能：

  • 訂閱用戶

  • 收藏視頻到 playlist

  • 下載普通視頻

  • 黑白主題切換

還未完成的功能:

  • 更加友好的進度條
  • 後臺播放
  • 下載動漫
  • 視頻轉碼成普通mp4格式
  • 刪除 playlist

還未修復的Bug:

  • 圖片有時加載失敗
